=================================
VIP TELEPORT MOD - By Saad3_3
=================================

[h1]OVERVIEW[/h1]
A premium, highly-configurable VIP Teleportation system for DayZ. Featuring multiple menu support, a modern glassmorphism UI, a robust anti-spam cooldown system, and restricted zone protection. Complete with a live in-game Admin Suite.

[h1]KEY FEATURES[/h1]
[list]
[*] [b]Advanced Cooldown System[/b] - Configurable wait times and hourly teleport limits to prevent abuse.
[*] [b]Restricted Zones[/b] - Define prohibited "Teleport FROM" areas (e.g., military bases or combat zones).
[*] [b]Modern Premium UI[/b] - Elegant glassmorphism aesthetic with animated progress bars and distance markers.
[*] [b]Smart Notification System[/b] - Replaces messy chat messages with high-end, color-coded UI notifications.
[*] [b]Live Admin Suite[/b] - Change cooldowns, toggle systems, and reload configs instantly without server restarts.
[*] [b]Automatic Menu Merging[/b] - Unified view for players with multiple permissions (Admins/VIPs/Public).
[*] [b]Vehicle Support[/b] - Teleport with vehicles and all passengers (fully configurable per menu).
[*] [b]Double-Click Teleport[/b] - Fast and intuitive navigation.
[*] [b]Dynamic Distance Display[/b] - View real-time distance to any teleport destination.
[*] [b]JSON-Powered[/b] - Extremely easy to manage through flexible config files.
[/list]

[h1]INSTALLATION[/h1]
[h2]Server Setup:[/h2]
[olist]
[*] Add [code]-mod=@VIPTeleport[/code] to your server launch parameters.
[*] Start the server to generate default config files.
[*] Go to: [code]DayZServer\Profiles\SOSH79\VIPTeleport\[/code]
[*] Edit [b]VIPTeleportLocations.json[/b] to create your menus and points.
[*] Add Admin Steam64 IDs to [b]VIPTeleportAdmins.json[/b].
[*] [b]Note:[/b] You can use the in-game Admin Panel to reload changes instantly!
[/olist]

[h1]USAGE[/h1]
[list]
[*] [b]Keybinds:[/b] Assign keys in Settings > Controls > SOSH79.
[*] [b]Main Menu:[/b] Browse locations, check distances, and double-click to teleport.
[*] [b]Admin Panel:[/b] Toggle cooldowns, set timers, and apply live updates via the Admin keybind.
[/list]

[h1]SAFETY & SECURITY[/h1]
[list]
[*] [b]Input Validation:[/b] Admin Panel prevents invalid numeric entries.
[*] [b]RPC Protection:[/b] Secure server-side validation for all teleport and config requests.
[*] [b]Logging:[/b] Dedicated session logs located in [code]Profiles\SOSH79\VIPTeleport\Logs\[/code].
[/list]

[h1]CONFIGURATION EXAMPLES[/h1]

[b]Multi-Menu JSON Logic:[/b]
[code]
{
    "CooldownSeconds": 30,
    "MaxTeleportsPerHour": 10,
    "EnableCooldown": 1,
    "Menus": [
        { "MenuTitle": "Public", "IsPublic": 1, "AllowVehicleTeleport": 0 },
        { "MenuTitle": "VIP", "IsPublic": 0, "AllowVehicleTeleport": 1, "AllowedSteamIDs": ["ID1", "ID2"] }
    ]
}
[/code]
